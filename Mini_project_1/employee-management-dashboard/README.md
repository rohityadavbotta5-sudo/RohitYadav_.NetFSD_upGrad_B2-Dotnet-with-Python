# Employee Management System

Project Name: Employee Management System
Submitted By: Akshata P Malghan
Batch: Batch 2

##  Overview

This is a frontend-based Employee Management System built using HTML, CSS, JavaScript, jQuery, and Bootstrap.
It allows admin users to manage employee records including adding, editing, deleting, searching, filtering, and viewing dashboard statistics.

---

##  Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* jQuery
* Bootstrap 5
* Bootstrap Icons
* Node.js (for project setup)
* Jest (for testing)

---

##  Dependencies

This project uses npm for dependency management.

### Installed Packages:

* **jest** (dev dependency for testing)

---

##  Setup Instructions

1. Extract the ZIP file
2. Open the project folder in terminal
3. Run the following command to install dependencies:

```
npm install
```

---

##  Run the Project

* Open `index.html` directly in the browser
  (No server required)

---

##  Run Tests (Optional)

```
npx jest
```
